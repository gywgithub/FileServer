## File Server Node 

> 一个 node 版文件服务器, 支持跨域, 图片在线预览

> 支持 post put 上传文件

#### 使用指南

克隆仓库

~~~
git clone https://github.com/gywgithub/FileServer.git
~~~

进入项目目录

~~~
cd FileServer
~~~

安装依赖

~~~
npm install
~~~

运行服务

~~~
node server.js
~~~

浏览器输入 http://localhost:3000/ 可以看到页面内容就OK了

浏览器输入 http://localhost:3000/images/loader.gif 查看图片
